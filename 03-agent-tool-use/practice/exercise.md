### 题1：给 Agent 添加新工具

**题目：** 给上面的 Agent 添加一个 `get_news(topic)` 工具，获取指定主题的最新新闻。

**提示：**
```python
def get_news(topic: str) -> str:
    """获取新闻（模拟）"""
    news = {"AI": "OpenAI 发布 GPT-5，性能大幅提升", "芯片": "台积电3nm产能紧张"}
    return news.get(topic, "暂无相关新闻")
```

### 题2：ReAct 流程分析

**题目：** 分析下面这个 Agent 执行过程，写出每一步的 Action 和 Observation：

用户问："北京到上海的距离是多少公里？"

模型回复：
```
🤔 我需要先知道两座城市的位置或直接查询距离。

🎬 Action: search_web("北京到上海直线距离")
👀 Observation: "约1080公里"

✅ Answer: 北京到上海直线距离约1080公里。
```

**参考答案：**
- Step 1: 思考（需要查距离）→ 行动（调用search_web）→ 观察结果（1080公里）
- 关键点：模型不是直接回答，而是**先想再查**，这减少了一步到位的错误概率。

### 题3：什么情况下需要工具调用？

**题目：** 判断以下问题，哪个需要工具调用，哪个不需要：

1. "今天北京多少度？"
2. "请把这段文字翻译成英文：你好世界"
3. "秦始皇统一六国是哪一年？"
4. "帮我查一下特斯拉最新股价"

**参考答案：**

| 问题 | 是否需要工具 | 原因 |
|------|------------|------|
| 今天北京多少度？ | ✅ 需要 | 实时数据，模型不知道今天 |
| 翻译成英文 | ❌ 不需要 | 模型本身就能翻译 |
| 秦始皇哪年统一 | ❌ 不需要 | 训练数据里有，可以直接答 |
| 特斯拉股价 | ✅ 需要 | 实时股价，模型数据有截止日期 |
