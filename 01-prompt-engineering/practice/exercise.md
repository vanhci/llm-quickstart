## 练习题

### 题1：改进客服 Prompt

**题目：** 用户的客服机器人有时候答非所问。请分析以下 Prompt 问题，并写出改进版本：

```python
# 当前 Prompt（有问题）
prompt = "你是客服，用户问：" + 用户问题
```

**提示：**
- 问题在哪里？（提示：缺少知识库注入、缺少格式要求）
- 改进方向：角色设定 + 知识库 + 输出格式限制

**参考答案：**

```python
def build_prompt(user_question: str) -> str:
    """改进版：明确角色 + 注入知识 + 约束格式"""
    prompt = f"""你是一个专业电商客服，回复要求：
1. 不超过3句话
2. 优先从知识库匹配答案
3. 知识库没有就说"这个我帮您转人工"

【知识库】
- 发货：24小时内发货，顺丰2-3天
- 退货：7天无理由，3日退款
- 尺寸：商品页有尺码表

用户：{user_question}"""
    return prompt
```

### 题2：设计 Few-shot 示例

**题目：** 用 Few-shot 让模型输出特定格式的回复，为这个任务设计示例：

任务：把中文名字转成拼音+首字母缩写
示例格式：`张三 → Zhang San (ZS)`

请写出完整的 Few-shot Prompt（包含 2-3 个示例）。

**参考答案：**

```python
prompt = """把中文名字转成拼音+首字母缩写。

示例：
李白 → Li Bai (LB)
王菲 → Wang Fei (WF)
刘亦菲 → Liu Yifei (LYF)

请转换：周杰倫"""
```

**原理：** 给模型看2-3个示例，它会自动照着格式模仿，比纯文字描述准确得多。
